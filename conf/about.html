About I'm safe

Purpose

The MIASMA (My I Am Safe Messaging App) is a simple web site/service that presents a form to all for the creation of Winlink/RRI/Quick Message email or SMS message. The app writes files that can be imported by PAT for transmission via radio.

Demonstration

This is only a DEMONSTRATION. I have no desire to support, maintain, update, add features. Source code is available on github and you are welcome to fork and modify.

Theory of Operation

  • Single Messages are created via the client and submitted via a web interface. This is BYOB (Bring Your Own Device)
  • The Miasma web server writes out a short file with the essentials (FromName, ToAddresses, and MessageText)
  • The Miasma server (non-web) sees that a file has been written to its inbox
  • The file is moved (and renamed to prevent collisions) to a pending folder
  • The file is read by an appropriate reader and a list of transfer (IASMessage) messages are created
  • That list is passed off to an instance of a Winlink Formatter (currently only PAT)
  • For each address in the addresses field, the formatter creates an appropriate message for delivery, via either email or SMS
  • A record of each accepted transfer message is written, and the file is moved to the outbox

Alternatively, the client can create a spreadsheet file, either Excel (xlsx or xls) or plain CSV and upload the file to the web server.

  • one header row
  • one message per row
  • three columns per message (FromName, ToAddresses, Text)
  • Multiple tabs are supported
  • row entries can be blank. If so, I pick up the value from the previous row. I call this feature auto-ditto and it is useful for sending a bunch of messages from the same FromName, or sending the same message text to a bunch of addresses (although you can also do this via commas, in the ToAddresses field)

Independently, a task is running on the host looking for files created by the formatter. When such files are found, the task starts an instance of the PAT Winlink client to transfer the message, via ham radio to a Winlink RMS gateway and delivery to the destination addresses.
