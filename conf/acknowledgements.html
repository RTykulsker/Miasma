Acknowlegements for I'm safe

Inspiration

On June 19, 2025, Dave Wilma, KG7LEA, gave a presentation to my local EmComm team on "I Am Safe" Welfare Messaging. The presentation showed how the Seattle Hubs, Radio Relay International and the Winlink development team got together to implement a system to send welfare messages using ham radio via Winlink.

MIRO

My local EmComm Team, MIRO, has been the driving force in getting me to create this demonstration.

  • Vadim Volk, N7PIX, for the initial suggestion to replace paper forms with client's personal devices and web server. Alas, Vadim wanted to demonstrate this in seven days for Field Day 2025
  • Carl Erikson, K7CGE, for suggesting Bootstrap as a web UI framework. If you don't like how the web interface looks today, imagine it without Bootstrap styling!
  • Woody Howse, K7EDH, for suggesting Excel files as an alternative method of accepting client messages
  • Mike Post, KJ7FIF, MIRO lead, for general support
  • Amanda Keverkamp, City of Mercer Island Emergency Manager, for constant support and pushing for a 2026 "version 2" demonstration.
